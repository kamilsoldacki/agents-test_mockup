QC ratings

LEGEND

Ship verdict — Top-line call: Would you ship this agent to a client in this language? Yes = ready; With reservations = usable but note caveats in comments; No = do not ship. This replaces any older Yes/No “Ship to client?” toggle.

Clusters — How it sounds = voice delivery (accent, pronunciation, prosody, artifacts). How it behaved = conversation dynamics (turn-taking, times you had to repeat, language integrity).

Scale — 1 = needs work, 2 = acceptable, 3 = native-quality (1–3 dimensions only).

Accent nativeness — 1 = clearly non-native throughout; 3 = indistinguishable from a native speaker. Example: persistent L2 rhythm on every sentence → 1.

Pronunciation — Numbers, prices, dates, proper names and brands (e.g. MyMagenta, Deutsche Telekom). 1 = systematic misreads; 3 = always correct. Example: “29,90 €” read as English twenty-nine ninety → 1.

Conversational naturalness — Prosody only: intonation, tempo, sentence endings (not turn-taking). 1 = flat / tutorial-like; 3 = natural spoken delivery. Example: every sentence ends with the same rising tone → 1.

Voice artifacts / loudness — Distortion, harsh compression, shouting, fatiguing volume. 1 = worst; 3 = clean and comfortable.

Turn-taking / responsiveness — Conversational dynamics: did the agent interrupt, leave awkward silence, reply late, or refuse barge-in? 1 = regularly broken timing; 3 = smooth, interruptible dialogue. Example: agent talks over you mid-question, or 3+ second dead air before answering → 1.

Times you had to repeat — Count of times you repeated yourself (ASR / hearing signal). Use 0 / 1 / 2 / 3+ — there is no separate 1–3 “understanding” score.

Language integrity — Stayed in the assigned language with no unwanted switching or foreign-accent bleed. 1 = frequent wrong-language slips; 3 = fully consistent. Example: German session drops English words or a US accent mid turn → 1.

One-off vs repeatable — In DETAILED mode, after Replay a sentence: tag whether the error came back (bug) or was a one-off slip.

Wording / content issue — Awkward or wrong phrasing only — not factual correctness or task success (those are out of scope). Leave a short example in the wording field.

Comments — Required for any score of 1: briefly say what went wrong. If you suggest a system prompt change, write why here. Also note caveats when the ship verdict is With reservations or No.

Scope (intentional) — Evaluate delivery, language quality, and conversational behavior — not factual correctness or whether the agent completed the customer task. Prices and product details are invented on purpose. Only review languages you are native or fluent in. Run 2–3 calls of 5–7 minutes each.

How to review

  1. Confirm you are reviewing a language you are native or fluent in — the language assigned for this QC.
  2. Open the legend next to the scores so the rating scale is clear, then start a call and speak with the agent as a customer would.
  3. Run 2–3 calls of 5–7 minutes each. Under How it sounds, score accent, pronunciation, prosody, and artifacts. Under How it behaved, score turn-taking and language integrity, and record Times you had to repeat (0–3+) — there is no separate “understanding” 1–3 score.
  4. Give a top-line ship verdict: Yes / With reservations / No — would you ship this agent to a client in this language?
  5. Scope (intentional): evaluate delivery, language quality, and conversational behavior — not factual correctness or task success. Prices and product details are invented on purpose. “Wording / content issue” is for awkward phrasing only.
  6. In DETAILED mode, if something is mispronounced, paste that exact sentence under Replay a sentence, play it again, and tag one-off vs repeatable bug.
  7. Leave a short comment whenever you score a 1 (and note caveats for With reservations / No).
