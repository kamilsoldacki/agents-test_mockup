How Productions runs a review

  1. Review the system prompt, first message, language, and LLM loaded live from the client agent — adjust only for this session if you need a controlled variant.
  2. Check voice settings (voice ID, model, stability, speed, similarity). Options reflect what the client configured; your edits stay session-only.
  3. Start a call and speak with the agent using the live session overrides from this page.
  4. For pronunciation QC, paste the same sentence into the TTS QC box and regenerate with the current voice settings to see if the issue repeats.
  5. Capture remarks for the client (aliases, wording, voice settings) — nothing here writes back to the agent.
